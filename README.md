# XMLib 动作游戏编辑器

- 逐帧动作编辑
- 逻辑与表现分离
- 动作逻辑实现简单
- 可用于帧同步
- 支持2D与3D

## 依赖

- [XMLib.Common](https://github.com/PxGame/XMLib.Common)

## 安装方式

 1. 克隆本库及其依赖库到工程中使用
 2. 如果你是git工程，则可将本库及其依赖库作为子库添加到你的git工程
 3. 通过 Unity Package Manager 的 Add Package from git URL 来添加
